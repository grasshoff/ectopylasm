Installation
************

To install ectopylasm, do:

.. code-block:: console

  git clone https://github.com/sundial-pointcloud-geometry/ectopylasm.git
  cd ectopylasm
  pip install .


Run tests (including coverage) with:

.. code-block:: console

  python setup.py test
